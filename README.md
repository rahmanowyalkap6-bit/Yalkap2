         H.E.R.O. - Аварийная Вертолетная Спасательная Операция
                        Описание игры
H.E.R.O. (Helicopter Emergency Rescue Operation) - это классическая аркадная игра, первоначально выпущенная компанией Activision в 1984 году для игровой консоли Atari 2600. В этой HTML5 адаптации вы берете на себя роль спасателя, оснащенного реактивным ранцем, чья миссия - спасти шахтеров, застрявших глубоко в подземной шахте.

Игровой процесс
Основная цель
Спасите всех шахтеров на каждом уровне
Избегайте врагов и опасностей
Собирайте фонари для дополнительных очков
Пройдите все уровни, чтобы завершить игру

            Управление
Стрелка влево/вправо - движение по горизонтали
Стрелка вверх - активация реактивного ранца для полета вверх
Стрелка вниз - ускоренное падение вниз
Пробел - альтернативная активация реактивного ранца
Элементы игры
            Персонаж
Оснащен реактивным ранцем для маневрирования в шахте
Может ходить по платформам и летать
Имеет 5 жизней в начале игры

           Шахтеры
Расположены в разных частях шахты
Излучают свет, что помогает их обнаружить
Каждый спасенный шахтер приносит 100 очков × текущий уровень

            Враги
Летающие существа, патрулирующие шахту
При столкновении отнимают одну жизнь
С каждым уровнем их количество увеличивается

       Платформы
Создают структуру шахты
На них можно стоять и передвигаться
Игрок может приземляться на них сверху

        Фонари
Разбросаны по шахте
Каждый собранный фонарь приносит 50 очков
Помогают увеличить общий счет

      Система уровней
Каждый уровень содержит определенное количество шахтеров
С каждым новым уровнем сложность увеличивается
Количество врагов растет с повышением уровня
Для перехода на следующий уровень нужно спасти всех шахтеров
Система подсчета очков
Спасение шахтера: 100 очков × текущий уровень
Сбор фонаря: 50 очков
Очки суммируются и отображаются в верхней части экрана

                           Историческая справка
Оригинальная игра H.E.R.O. была разработана Джоном Ван Райзем и выпущена в марте 1984 года. Она получила признание за инновационный геймплей, сочетающий элементы платформера и головоломки. Игра была портирована на множество платформ того времени, включая Commodore 64, ZX Spectrum и MSX.
Данная HTML5 версия сохраняет дух оригинала while используя современные веб-технологии для воссоздания классического игрового опыта в браузере.

           Управление сложностью
Игра начинается с относительно простого первого уровня, где представлены основные механики. По мере прохождения:
Увеличивается количество врагов
Усложняется структура шахты
Требуется более точное управление реактивным ранцем
Увеличивается количество шахтеров для спасения
Эта прогрессивная система сложности обеспечивает плавную кривую обучения и поддерживает интерес игрока на протяжении всей игры.
